# Citadel Universe website

This website is the official focal point for all information about the 3D FPS Amiga game "Citadel" originally released in 1995, its later remakes, followups, offshoots, art, stories - the whole universe related to the Citadel and its lore.


# Website development environment

- VS Code
- Live Preview plugin (click "show preview" icon at the right top corner of the html page open in the editor and connect external browser to http://localhost:3000/)
- If there are 401 authentication errors loading some elements in an external browser (e.g. bootstrap fonts/icons), just restart VS Code
